Logger Test in Browser

This page tests the logger utility in a browser environment. Open your browser's developer console (F12 or Ctrl+Shift+I) to see the log messages.

The test will automatically run when the page loads, but you can also click the button below to run it again.

Note: In Chrome-based browsers, you need to enable "Verbose" log level in the console to see all debug messages.

To do this, click on the "Default" or "Info" dropdown in the console and select "Verbose".
